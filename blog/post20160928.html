Corrigindo bug "ASP.NET add-in not found" do MonoDevelop no Ubuntu 16.04

Primeiro, você precisa adicionar a chave GPG e o repositório de pacotes do projeto Mono ao seu sistema, conforme instruções aqui.

Depois, você precisa "pinar" o pacote para não instalar do repositório do Ubuntu, que oferece uma versão mais alta. Como root, abra o arquivo /etc/apt/preferences e inclua o seguinte conteúdo: 

Package: monodevelop
Pin: origin archive.ubuntu.com
Pin-Priority: -1

Salve e rode:

sudo apt update
sudo apt install monodevelop=5.10.0.871-0xamarin2
