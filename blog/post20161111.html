Instalando PostgreSQL 9.5 ou 9.6 no Debian Jessie

Você deve saber que o Debian Jessie oferece o PostgreSQL 9.4 no seu repositório padrão. Mas é possível instalar o PostgreSQL 9.5 ou até mesmo o 9.6 no seu Debian.

O primeiro passo é garantir que o seu Debian está atualizado. Como root, rode: 

apt update
apt upgrade

Agora abra o arquivo /etc/apt/sources.list com o seu editor de texto favorito e inclua o seguinte conteúdo: 

# PostgreSQL repository
deb http://apt.postgresql.org/pub/repos/apt/ jessie-pgdg main

Salve e rode:

wget --quiet -O - http://apt.postgresql.org/pub/repos/apt/ACCC4CF8.asc | apt-key add -
apt update
apt install postgresql-9.5 # ou postgresql-9.6

Para fazer o PostgreSQL iniciar no boot:

update-rc.d postgresql defaults

Não esqueça de alterar a senha do usuário postgres:

su postgres
psql
\password postgres
\q
exit

Se você quer acessar o PostgreSQL de outro computador, abra o arquivo /etc/postgresql/9.5/main/postgresql.conf e altere o seguinte parâmetro:

listen_addresses = '*'

Agora abra o arquivo /etc/postgresql/9.5/main/pg_hba.conf e adicione a seguinte linha:

host all all 192.168.0.0/24 md5

Por último, rode o comando abaixo como root para reiniciar o PostgreSQL:

service postgresql restart
