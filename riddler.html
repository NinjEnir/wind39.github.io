Projeto Riddler!

Riddler é um programa que criptografa e descriptografa arquivos e mensagens.

Implementa o algoritmo de Rijndael, com criptografia forte de 256 bits, que só pode ser descriptografada com a mesma senha que foi usada na hora de criptografar.

O projeto foi construído para testar dois namespaces da biblioteca Spartacus:

  • Spartacus.Forms: Permite o desenvolvimento rápido de formulários e telas simples, sem a necessidade de usar designer de interfaces. Os componentes foram projetados para serem bem fáceis de configurar.
  • Spartacus.Net: Suporte para desenvolver aplicativos que se comunicam pela rede. Esquema cliente e servidor e definição de pacotes de dados. Criptografia.

Curitiba, 23 de fevereiro de 2015.
