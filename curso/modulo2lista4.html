Lista de Exercícios Módulo II Capítulo 4

1) Houve uma mudança de requisitos no projeto da escola. Agora é necessário que, para cada turma, haja um controle de faltas e notas por aluno. Para isso, você precisará alterar a classe Turma para atender a este novo requisito, e também modificar o menu para mostrar essa nova operação.

2) CONTROLE DE ESTOQUE. Uma empresa que compra e revende uma determinada mercadoria (à sua escolha) contratou você pra desenvolver um sistema de controle de estoque, que deve ser capaz de manipular:

  • Cadastro de Mercadorias
  • Estoque de Mercadorias
  • Cadastro de Fornecedores
  • Compras de Mercadorias (entradas no estoque)
  • Cadastro de Clientes
  • Vendas de Mercadorias (saídas de estoque)

3) GESTÃO DE BIBLIOTECA. Você foi contrado(a) para desenvolver um sistema para gestão de uma biblioteca, com os seguintes requisitos:

  • Cadastro de Livros
  • Cadastro de Revistas
  • Cadastro de Fitas de Vídeo
  • Cadastro de Usuários
  • Empréstimos
