Módulo I Avaliação 1

0) Leia atentamente os critérios para avaliação. Ao concluir cada exercício, mande a pasta do projeto compactada ou apenas o código-fonte para william.ivanski@gmail.com. Inclua seu nome no assunto ou no corpo do e-mail para que eu possa identificá-lo.

1) Escreva um programa que solicite ao usuário uma sequência de números reais. O usuário poderá digitar quantos números reais quiser; quando quiser parar de digitar, o usuário digitará 0 (zero). Então, o programa deverá mostrar a média aritmética dos valores digitados pelo usuário (exceto o número zero, que não deverá ser contabilizado na média).

2) Escreva um programa que solicite ao usuário o valor de N, um número natural. N será a altura do triângulo retângulo a ser desenhado na tela pelo seu programa, usando apenas o caractere * (asterisco). Por exemplo, para N = 7, o programa deverá imprimir: 

*
**
***
****
*****
******
*******
