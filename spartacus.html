Biblioteca Spartacus   

É com muita satisfação que eu informo a todos que publiquei a minha biblioteca Spartacus no GitHub!!

Para quem não sabe, o GitHub é o maior repositório de projetos de código aberto do mundo. Hospeda o código-fonte de projetos importantes, como o Mono, o JQuery e até mesmo o kernel do Linux, que pertence a Linus Torvalds, o criador do Linux.

A Spartacus é uma biblioteca em C# que venho desenvolvendo há quase 4 anos. Comecei a desenvolver em casa por hobbie, quando percebi que estava tendo que escrever muito código repetido para tarefas semelhantes no meu dia-a-dia. O código todo já chegou a ter mais de 20 mil linhas, e então eu separei em duas partes: criei uma segunda camada somente com código de manipulação de arquivos NF-e, SPED e outras expertises do meu emprego. Essa segunda camada não é pública, pertence ao meu contratante.

A primeira camada da Spartacus, que estou abrindo sob a licença MIT, tem cerca de 12 mil linhas de código, e se beneficia de algumas outras bibliotecas também de código aberto. Abrir a Spartacus foi a forma que eu encontrei de retribuir à comunidade e respeitar suas licenças e seus créditos.

Por enquanto, a Spartacus possui os seguintes módulos:

  • Spartacus.Database: Contém encapsulamentos para acessar diversos bancos de dados diferentes de forma semelhante. Exemplos de SGBDs suportados: Firebird, PostgreSQL, MySQL, SQLite, Oracle (via ODBC) e muitos outros via ODBC e OLEDB.
  • Spartacus.Forms: Permite o desenvolvimento rápido de formulários e telas simples, sem a necessidade de usar designer de interfaces. Os componentes foram projetados para serem bem fáceis de configurar.
  • Spartacus.Net: Suporte para desenvolver aplicativos que se comunicam pela rede. Esquema cliente e servidor e definição de pacotes de dados.
  • Spartacus.Reporting: PDF Report Engine, permite renderizar relatórios em PDF a partir de um arquivo XML que contém a definição do layout do relatório e dados extraídos com o módulo Spartacus.Database.
  • Spartacus.Utils: Classes para extração de dados de arquivos do Microsoft Excel (CSV, XLS e XLSX) e outras coisas.

O projeto já está na versão 0.30.0, que em termos de comentários está completamente documentada. Ainda preciso escrever um wiki no GitHub, elaborar manuais de utilização, definição do layout XML para geração de relatórios em PDF... Enfim, o projeto está longe de ficar pronto, pelo contrário, agora é que está começando.

Criei uma página específica para a Spartacus aqui no site, e disponibilizarei aqui mesmo os binários compilados para serem baixados gratuitamente. Você até já pode baixar a versão 0.30.0!!

Bom, o fato é que a Spartacus agora está disponível no GitHub. O código-fonte é aberto sob a licença MIT, o que significa que você pode baixar e usar em seu aplicativo comercial. Esse é o primeiro projeto de código aberto de minha autoria, e ainda estou aprendendo como esse mundo fascinante funciona!

Curitiba, 11 de setembro de 2014.
