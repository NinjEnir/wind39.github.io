OmniDB 1.3, agora com Conversão de Schema entre tecnologias!   

Pessoal, é com grande satisfação que anuncio a versão 1.3b do OmniDB, agora com conversão de schema entre tecnologias!

Dentre outras novidades, agora é possível realizar conversão entre qualquer SGBD já suportado pelo programa (são 7 no momento) e é muito simples de fazer.

Pra quem ainda não conhece, OmniDB é uma ferramenta web grátis brasileira pra gerenciar banco de dados. A nossa equipe desenvolveu ela sob medida pra minimizar as dificuldades mais comuns encontradas no mercado, solucionando problemas reais de empresas em que atuamos.

Você pode baixar a nova versão no site oficial.

Aqui estão as alterações da nova versão:

  • Conversão de schema entre tecnologias;
  • Visualização DER completo;
  • Histórico de comandos;
  • Suporte a "NT Authentication" no SQL Server;
  • Pequenas correções de bugs.

Curitiba, 11 de abril de 2016.

OmniDB agora suporta Microsoft SQL Server   

Você pediu, nós atendemos! Hoje estamos felizes em anunciar que o OmniDB agora suporta Microsoft SQL Server.

As conexões com SQL Server funcionam como no PostgreSQL, em que é necessário se conectar a um schema específico. Porém, no SQL server, se deixar o schema em branco, o OmniDB se conectará automaticamente ao schema padrão "dbo".

Estamos aguardando ansiosamente pelo retorno da comunidade. Enquanto isso estamos arregaçando as mangas e corrigindo pequenos bugs, melhorando a interface com o usuário e adicionando novas funcionalidades e suporte a novos SGBDs.

Fique ligado para novidades!

Curitiba, 26 de fevereiro de 2016.
